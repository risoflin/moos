uf_save_dw_as_excel() function

Converts datawindow into excel file

Syntax

uf_save_dw_as_excel ( dw, filename )

Parameters

dwA reference to the datawindow object
filenameA string whose value is the name of the file you want to create. If filename is not on the operating system's search path, you must enter the fully qualified name.

Return value

Integer. Returns 1 if it succeeds and -1 if an error occurs.

Examples 

integer li_ret

li_ret = uf_save_dw_as_excel(dw_1, 'c:\doc\report.xls')
if li_ret = 1 then
   //success
   //...
else
   //fail
   //...
end if
