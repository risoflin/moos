of_save_dw2sheet() method

Converts datawindow to new Sheet of Workbook.

Syntax

of_save_dw2sheet (datawindow, parm)

Parameters

datawindowA reference to the datawindow object
parmA reference to the n_dwr_service_parm object which contain a set of parameters for process of converting

Return value

Integer. Returns 1 if it succeeds and -1 if an error occurs. If argument value is NULL, of_save_dw2sheet returns NULL.

Examples 

n_dwr_service_parm lnvo_parm
n_dwr_service_excel lnvo_excel

lnvo_excel = create n_dwr_service_excel 

lnvo_parm = create n_dwr_service_parm
lnvo_parm.ib_show_progress = true
lnvo_parm.is_version = '97'
lnvo_parm.ib_group_header  = true

//Create Workbook
if lnvo_excel.of_create_wb('C:\MY_BOOK.XLS', lnvo_parm) = 1 then
	
	//Converting Datawindow to Sheet "Example 1"
	lnvo_parm.is_sheet_name = 'Example 1'
	lnvo_excel.of_save_dw2sheet(dw_1, lnvo_parm)
	
	//Converting Datawindow to Sheet "Example 2"
	lnvo_parm.is_sheet_name = 'Example 2'
	lnvo_excel.of_save_dw2sheet(dw_2, lnvo_parm)
	
	//Converting Datawindow to Sheet "Example 3"
	lnvo_parm.ib_group_header  = false
	lnvo_parm.is_sheet_name = 'Example 3'
	lnvo_excel.of_save_dw2sheet(dw_3, lnvo_parm)

	//Close Workbook
	lnvo_excel.of_close_wb()
end if
